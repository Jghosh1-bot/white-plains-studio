# White Plains Studio - Professional Rental Website

A modern, high-quality website for the White Plains Premium Fitness & Martial Arts Studio rental space.

## Features

✅ **Professional Design** - Modern, clean layout matching current web standards  
✅ **Fully Responsive** - Works perfectly on mobile, tablet, and desktop  
✅ **Fast Load Times** - Minimal code, optimized performance  
✅ **SEO Ready** - Proper headings, metadata, and structure  
✅ **Interactive Elements** - Hover effects, smooth scrolling, animations  
✅ **Contact Form** - Easy inquiry submission  
✅ **Mobile Menu** - Hamburger menu for small screens  

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations
- `images/` - Folder for studio photos

## How to View the Website

1. Once files are uploaded, enable GitHub Pages
2. Go to Settings → Pages
3. Select `main` branch
4. Your site will be live at `https://jghosh1-bot.github.io/white-plains-studio`

## How to Customize

### Change Colors
Edit the `:root` variables in `styles.css`:
- `--primary-red: #DC143C;` - Main brand color
- `--primary-dark: #1a1a1a;` - Dark background
- `--secondary-gray: #f5f5f5;` - Light background

### Update Contact Information
Edit the contact section in `index.html` with your details

## Deployment

### Netlify (Recommended)
1. Go to [netlify.com](https://netlify.com)
2. Connect your GitHub repository
3. Your site is live!

### GitHub Pages
1. Go to Settings → Pages
2. Select `main` branch
3. Your site will be live automatically
